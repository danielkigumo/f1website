# F1 World — 2026 Season Tracker

Web Application Development project, Strathmore University, June 2026.  
Built by **Daniel Kigumo**.

## What it is

An 8-page F1 fan site covering the 2026 Formula 1 season. Driver profiles, team info, championship standings, the full race calendar, and a news section. I chose F1 because I followed the sport for a while and it has enough real data to make an interesting project.

## Pages

| File | Page |
|------|------|
| `index.html` | Home — hero, stats, and championship leader |
| `drivers.html` | All 20 drivers with live search/filter |
| `teams.html` | All 10 constructor teams |
| `news.html` | Season news stories |
| `standings.html` | Drivers and constructors championship tables |
| `races.html` | Full 24-race calendar with status badges |
| `about.html` | About the project and what F1 is |
| `contact.html` | Contact form with JS validation |

## Technologies

- HTML5 (semantic tags, proper structure)
- CSS3 (custom stylesheet on top of Bootstrap)
- Bootstrap 5.3 (grid, navbar, responsive layout, tables)
- JavaScript ES6+ (three interactive features)

## JavaScript Features

1. **Contact form validation** — checks name, email format, subject selection, and message length before submission
2. **Dark / light mode toggle** — persists across page navigation using localStorage
3. **Live driver search** — filters the 20 driver cards in real time as you type (by name or team)
